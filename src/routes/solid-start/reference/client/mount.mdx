---
title: mount
---

Method that calls either [`hydrate`](/reference/rendering/hydrate) (server rendering) or [`render`](/reference/rendering/render) (client rendering) depending on the configuration. It's used in [`entry-client.tsx`](/solid-start/reference/entrypoints/entry-client) to bootstrap the application.

```tsx twoslash
import { mount, StartClient } from "@solidjs/start/client";

mount(() => <StartClient />, document.getElementById("app")!);
```

If you set `{ ssr: false }` in the [`defineConfig`](/solid-start/reference/config/define-config), thus entirely deactivating hydration, then mount becomes equal to [`render`](/reference/rendering/render).

## Parameters

| Prop | type              | description                                 |
| ---- | ----------------- | ------------------------------------------- |
| fn   | () => JSX.Element | Function that returns the application code. |
| el   | MountableElement  | DOM Element to mount the application to     |
