---
title: "StartServer"
---

`StartServer` takes a function returning a document component which will serve as the static document for your application. It's used in the [`entry-server.tsx`](/solid-start/reference/entrypoints/entry-server) for bootstrapping the server.

```tsx twoslash
import { StartServer } from "@solidjs/start/server";
```

## Reference
### `<StartServer />`


| Prop      | Type     | Description                                                                                   |
| --------- | -------- | --------------------------------------------------------------------------------------------- |
| document  | Function | A function that returns the static document for your application.                             |
