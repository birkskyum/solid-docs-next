title: app.config.ts
---

The `app.config.ts` in the root of every Vinxi app, and thus also every SolidStart app, is the main point of configuration. This file should export a configuration for SolidStart, [Vinxi](https://vinxi.vercel.app/), [Vite](https://vitejs.dev) and [Nitro](https://nitro.unjs.io). The easiest way to generate a configuration is to use the [`defineConfig`](/solid-start/reference/config/define-config) helper.

Here [`defineConfig`](/solid-start/reference/config/define-config) is used to create a minimal configuration with default settings.

```tsx
import { defineConfig } from "@solidjs/start/config";

export default defineConfig({});
```

import { defineConfig } from "@solidjs/start/config";

For more configuration options see [`defineConfig`](/solid-start/reference/config/define-config).
